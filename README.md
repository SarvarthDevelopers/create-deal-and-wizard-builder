# Create Deal and Wizard Builder

A premium interface for capturing deal information and building dynamic wizards.

## Features
- **Create Deal Flow**: A streamlined modal for creating pawn or purchase deals.
- **Specialized Car Flow**: Advanced VIN search simulation, searchable make/model dropdowns, and automated item naming.
- **Wizard Builder**: An interactive drag-and-drop interface for configuring multi-step wizards with custom fields.
- **Live Calculation Engine**: Real-time summary and validation of deal values and LTV.

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Animations)
- React DnD (Drag & Drop)

## Getting Started
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`
